MusicEvent: Big Data & Semantic Web application for music

During my master's learning in Big Data and Knowledge management, our team realized a small web application for music fans to help them find coming music festivals near their city. Inspired by big data technologies and powerful AngularJS, we have chosen SOA architecture and integrated varies web resources.

Overview

Inspired by big data trends and meets the needs of project, our team realized this small application to help people find music events that will be held near the city where they live. To achieve a good coverage of necessary information, we have integrated with many famous semantic web resources like DBPedia, jamendo and musicbrainz

Our project contains mainly 3 parts: Web design and front-end realization, web resources integration and SOA web service development. During the project, I realized front-end technologies under AngularJS framework and provided RESTFul web services using Jax-rs.


Semantic web and related technologies

What is semantic web? Semantic web is considered as a standard which promote common data across the world wide web. With semantic web, which is also a component of Web 3.0, we share and connect separated resources together to meet our needs and reuse them.

Among related technologies of semantic web, we chose Jena framework(Apache) to realize semantic web architecture, SPARQL and resource integration. Highly motivated by AngularJS, we pursued the idea of SOA and finally came out our system structure: AngualrJS+JAX-RS+Jena+EclipseLink+Mysql. To store semantic web data locally, we used TDB (Triple Store) provided by Jena framework.
